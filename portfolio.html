Projects

I have undertaken several projects, some as part of my undergraduate studies, others as part of my co-curricular activities. Please have a look!

2023: intUItive! - VR Project

This was done as part of a final project for a class on Virtual Reality interaction design. intUItive! explores user interface/user experience design in VR. I developed parts of the Unity map, scripts, and the overall design of the project.

2023: MovieMate - Software Engineering Project

I worked with a team to create a simple Command-Line application in Java as part of a software engineering module. In the process, I learned Object-Oriented Programming practices and how to use GitHub to work on software in a team.

2022: Bootstrap Bootcamp

I spearheaded and planned a one-day introductory workshop on creating a website using HTML, CSS, and Bootstrap. Participants were fellow schoolmates from NUS. During the event, they made their own website to store their portfolio and CV (Much like this one!), and networked with each other as well. This workshop was held as one of the NUS Communications and New Media Society's events, and I had assistance from fellow members and staff advisor with administrative details. (Photos TBA)

2021: Parasite - Game Design Project

The video showcases our game, Parasite, made with Game Maker Studio 2. It was done as a final project for a module on Game Design. I was the lead programmer and producer, programming the source code, helping in other aspects like art, while managing the team and ensuring we met our deliverables. Major props to my teammates for this wonderful game made in seven weeks!

2021: Bounce

This was part of my final project for a module on HTML, CSS, and Javascript. For this project, I created an interactive ball physics simulator.
Additionally, I used Node.js to implement a dynamic web page where multiple users could play a "ping-pong" game using similar ball physics. Unfortunately, GitHub can only serve static pages so that is not shown.

2021: Media Release

As the final assignment of a module on media writing, I had to produce a press release and social media collaterals for a fictional campaign.
Content was created and put together by me, with images sourced online (For the purpose of this assignment). Although this is a fictional client, I've included this here as a sample of my media writing and graphic design work.

2021: Feature Article

In the same module on media writing, I also produced a feature article on a fellow schoolmate's volunteering initiative. Photos and article are original.

Graphic Design & Digital Painting

A small selection of images I have edited on Adobe Photoshop for personal or school-related collaterals. Included in the gallery are also some digital paintings I've done. I've been doing image editing and digital/traditional art since young, self-taught using tutorials.

Videography

I've picked up Adobe Premier Pro and Adobe After Effects for personal projects, mainly videos made for my friends or school events. They act as a means for me to practice and explore the capabilities of video editing.
