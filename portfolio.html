Projects

I have undertaken several projects, some as part of my undergraduate studies, others as part of my co-curricular activities. Please have a look!

Bootstrap Bootcamp

I spearheaded and planned a one-day introductory workshop on creating a website using HTML, CSS, and Bootstrap. Participants were fellow schoolmates from NUS. During the event, they made their own website to store their portfolio and CV (Much like this one!), and networked with each other as well. This workshop was held as one of the NUS Communications and New Media Society's events, and I had assistance from fellow members and staff advisor with administrative details.

2022: Bounce

This was part of my final project for a module on HTML, CSS, and Javascript. For this project, I created an interactive ball physics simulator.
Additionally, I used Node.js to implement a dynamic web page where multiple users could play a "ping-pong" game using similar ball physics. Unfortunately, GitHub can only serve static pages so that is not shown.

2021: Media Release

As the final assignment of a module on media writing, I had to produce a press release and social media collaterals for a fictional campaign.
Content was created and put together by me, with images sourced online (For the purpose of this assignment). Although this is a fictional client, I've included this here as a sample of my media writing and graphic design work.

2021: Feature Article

In the same module on media writing, I also produced a feature article on a fellow schoolmate's volunteering initiative. Photos and article are original.
